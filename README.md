# TRAIN SCHEDULE APPLICATION

## LINK TO LIVE GAME: https://rubyrgill.github.io/Train-Schedule/

## Application Objective: 
```
 - Using Firebase, push details of new train to database 
 - Pull information from database and display in table 
 - Calculate next train and minutes away on submit
```
## Tools/Programs Used: 
```
 - Firebase database 
 - Javascript (Jquery) 
 - GoogleFonts
 - HTML/CSS
 - Momentjs
```
## Application Instructions

### Initial Screen:
```
 - Application displays table of previously entered details of trains from all users
 - Displays on all machines
```
