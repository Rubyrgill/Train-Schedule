# TRAIN SCHEDULE APPLICATION

## LINK TO LIVE GAME: 

## Application Objective: 
```
 - Using Firebase, push details of new train to database 
 - Pull information from database and display in table 
 - Calculate next train and minutes away on submit
```
## Tools/Programs Used: 
```
 - Firebase database 
 - Javascript (Jquery) 
 - GoogleFonts
 - Momentjs
 - HTML/CSS
```
